Estrutura Condicional Composta em Java

Este repositório apresenta exemplos práticos utilizando estruturas condicionais compostas em Java.
Foram desenvolvidos dois exercícios simples e didáticos:

-Verificação de elegibilidade para votação (if / else)
-Identificação de animais pela quantidade de pernas (switch-case)

O objetivo do projeto é treinar lógica de decisão e entender como o Java interpreta diferentes caminhos dentro de um programa.
